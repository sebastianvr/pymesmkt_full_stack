El mapa de colaboraciones es un grafo generado automáticamente a partir de las compras y ventas realizadas por las propias empresas.

Un grafo se compone de un conjunto de vértices y un conjunto de aristas (o enlaces) que conectan pares de vértices.

Cada nodo, representado por un círculo azul, simboliza una empresa, con el nombre de la empresa ubicado debajo.

Cada vértice tiene asociado un peso, un valor numérico que representa la cantidad de transacciones hechas de una empresa a otra.

Funcionalidades disponibles:

  • Zoom: Puede hacer zoom utilizando los botones ubicados en el extremo superior izquierdo del gráfico. También puede hacer zoom utilizando el rodillo del mouse o haciendo doble clic en el área del grafo.
  • Mover nodos: Para mover un nodo, selecciónelo con un clic y arrástrelo al lugar deseado.
  • Centrar grafo: Use el botón en el extremo superior derecho para centrar el grafo cuando sea necesario.
  • Información de nodos: Para ver un nodo específico junto con sus vecinos, pase el cursor sobre el nodo deseado. Esto resaltará el nodo seleccionado junto con sus nodos vecinos y las conexiones entre ellos. Los colores de las conexiones indican el tipo de transacción: verde para compras y azul para ventas.
