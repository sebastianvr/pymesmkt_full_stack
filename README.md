# Para ejecutar en desarrollo

  1. Clonar repo
  2. Ejecutar comando
  
  ```bash
    $ yarn install
  ```
  3. Tener Nest CLI instalado

  ```bash
    $ yarn global add @nestjs/cli
  ```
  4. Levantar la base de datos

  ```bash
    $ docker-compose up -d
  ```
  5. Para reconstruir la BD
  
  ```
     http://localhost:3000/api/v2/seed
  ```
